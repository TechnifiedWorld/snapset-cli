# Security Policy

## Scope

SnapSet is a Windows batch script launcher. It opens built-in Windows settings pages via `ms-settings:` URIs and legacy Control Panel shortcuts. It does not handle user data, network traffic, credentials, or external input beyond a single command argument.

## Reporting a Vulnerability

If you discover a security issue such as argument injection or unintended system access, please do not open a public issue.

Report it directly to the maintainer via GitHub:
[https://github.com/TechnifiedWorld](https://github.com/TechnifiedWorld)

Please include:

- A description of the issue
- Steps to reproduce it
- The Windows version and build number you tested on

All reports will be reviewed and responded to promptly.
